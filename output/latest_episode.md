# 2026年06月14日 播客脚本

【第一部分：核心持仓点评】

[阿哲] 好,今天是2026年6月14日,周日——美股已经收盘了上一个交易日,我们来复盘本周末这批持仓的表现。今天这张单子里,动静最大的毫无疑问是RKLB,直接砸了将近11%。

[小晴] 对,RKLB这个跌法其实我在周五盘中就盯着了。从标题来看,催化剂非常明确——SpaceX正式以SPCX为代码在纳斯达克挂牌交易,当天一堆空间赛道的公募股都跟着集体大跌。资金的逻辑很直白:投资者在SpaceX这个赛道"旗舰"上市的那一刻,选择旋转出公开市场的小弟们。

[阿哲] 这个逻辑我理解,但我觉得市场的反应有点过度。你看基本面,Rocket Lab刚刚发布了Q1 2026创纪录的营收2.0035亿美元,同比增长约64%,订单积压已经到了22亿美元,Q2指引也给到了2.25到2.4亿美元之间。这不是一家经营在崩塌的公司。

[小晴] 但市场有时候就这样——情绪先行。而且这次跌法也有结构性压力。公司最新财报虽然营收创新高、积压大幅增长,但市场同时在消化毛利率下行的预期,原因是订单结构向大型政府合同倾斜——主要是美国太空发展署的合约。短期内这个逻辑会压制估值修复。

[阿哲] 对,而且从历史波动来看,过去10个交易日里RKLB已经累计下跌了将近29%,从历史高位150美元区间回撤幅度非常大。接下来1-2个交易日,我会持观望态度——SpaceX挂牌后的资金流向是否重新回补,是关键看点。如果SPCX上市后出现"买预期卖事实"的反转,RKLB反而可能有技术性反弹机会。

[小晴] 而且别忘了,Neutron火箭的首飞进展依然是最核心的催化剂,这个milestone的兑现时间将直接影响RKLB中期估值。讲完RKLB,我们说说LLY。今天跌了2.41%,我注意到这两天LLY情绪不太好。

[阿哲] 驱动因素上周就埋下了。路透社的报道显示,有雇主正在考虑在2027年取消对GLP-1减肥药的报销覆盖,这个消息让LLY股价周四就承压了。Damodaran讲过,大盘医药股的估值里,市场对"可及性"的定价非常敏感——支付方收紧覆盖面,等于直接打击市场规模假设。

[小晴] 但基本面我觉得依然扎实。上个季度LLY的EPS是8.55美元,大幅超出华尔街预期的6.85美元,比去年同期的3.34美元翻了一倍多。而且公司在Q1发布后已经上调了全年2026年的营收指引,主要驱动来自Mounjaro和Zepbound的超预期表现。

[阿哲] 是的,所以这次跌法更多是情绪扰动而非基本面恶化。我的框架里,LLY现在处于"高质量资产的调整窗口"——P/E大概40倍,虽然Novo Nordisk是GLP-1注射剂的先行者,但LLY后来居上,已经在市场份额上超越。接下来1-2个交易日,如果没有新的医保政策负面信号,股价应该会在当前区间企稳。

[小晴] 好,接下来说PLTR。今天跌了2.34%,但我觉得这个跌幅更多是估值在高位的自然回摆。从基本面看,Palantir Q1 2026的营收是16亿美元,同比增长将近85%——这个增速本身其实非常强劲。

[阿哲] 增速强没错,但估值层面的争议从来没有消失。有分析师给出了225美元的买入目标价,但12个月综合目标价中位数大概在184美元,当前128美元的股价相对这个水平其实已经有折价。问题在于市场给的倍数一旦收缩,中间的空间就很危险。

[小晴] CEO Alex Karp这几天的发言在社媒上引发了不少讨论。他公开说企业客户对AI前沿大模型厂商的销售和部署方式越来越不满——言下之意是Palantir的企业AI工具是更务实的选择。这个叙事对散户情绪有一定支撑作用,但机构该怎么定价还是另一回事。短期看,PLTR如果守住125美元附近支撑位,应该能继续震荡盘整。

[阿哲] 下面快过一下MU——今天跌了1.43%。关键催化剂在于Micron会在6月24日公布Q3财报,进入财报前往往有"做空对冲"资金涌入。但分析师情绪其实偏正面,Wolfe Research把目标价从550美元大幅上调到了1250美元,维持跑赢评级。这个目标价相对于当前981美元还有很大空间,市场可能在等财报做最后验证。

[小晴] 对,MU就是典型的财报前观望盘——方向不明就先缩量。接下来24日是关键节点,市场会重点盯AI存储需求,HBM芯片的订单情况一旦超预期,股价反应会很剧烈。

[阿哲] 然后是TSLA——今天是持仓表里唯一上涨超过1%的,涨了1.83%。马斯克身家随着SpaceX IPO的大涨越过了万亿美元,SpaceX估值飙升至将近2.2万亿美元,市场对此有连锁反应。TSLA某种程度上也在"蹭"这波马斯克光环。

[小晴] 我在X上看到有人专门讨论这个——SpaceX市值飙上去之后,有人反过来觉得TSLA估值有点"相对便宜"了。这种情绪驱动的逻辑听起来荒谬,但短期内确实有效。技术面上TSLA今天也守住了400美元的心理关口,接下来如果能站稳上方,有可能持续到周一。

[阿哲] 好,我们快速扫一下其余标的。TSM今天小涨0.68%,相对平稳。台积电是AI算力产业链的"地基"——上游需求不减,股价走势自然有韧性。MU财报前市场对HBM需求的预期也在间接给TSM撑盘。这块我觉得逻辑比较干净,没有特别值得担心的短期风险。

[小晴] META今天只跌了0.25%,基本算横盘。广告市场没有明显坏消息,AI基础设施投入的叙事依然是主线,市场对META的中期展望相对共识,没什么好说的。AMZN今天跌了1.22%,更多是大科技板块的系统性压力带动,AWS增长逻辑没有变化。

[阿哲] AVGO跌了0.9%,MRVL跌了0.35%——这两个都是AI定制芯片赛道,跟NVDA今天基本持平+0.17%的走势对照来看,整个AI芯片板块今天属于横盘震荡。NVDA现在205美元附近,从高点回落比较多,市场在等下一个产品周期催化。

[小晴] ORCL今天几乎没动,+0.03%。Oracle云和AI数据库的叙事市场消化得比较充分了,短期没有新的catalyst,这个票就会这样沉默地趴着。MSFT也是+0.11%,Copilot的商业化数据仍是机构持续关注的核心变量,但今天没有增量信息。

[阿哲] AAPL跌了1.52%,有观点认为苹果在AI资本支出上的保守态度,在AI股票整体回调的背景下反而显得更"聪明"。但短期内iPhone的AI叙事落地进展还是决定估值方向的关键。我会把AAPL留在观察名单,等52周新高或者WWDC后续落地产品的进展。

[阿哲] EQQQ今天数据暂时没有,但这是追踪纳斯达克100的欧洲ETF,今天纳指整体相对平稳,EQQQ走势基本与之对应,不会有太大异常。

【第二部分：市场热门股扫描】

[小晴] 好,第二部分来说说今天美股市场和社媒上讨论度很高、但不在我们持仓单里的票。第一个当然是SPCX——SpaceX。这绝对是本周最大的故事。

[阿哲] 对,SpaceX在纳斯达克正式挂牌交易,代码SPCX,这次IPO规模打破了沙特阿美2019年290亿美元的历史纪录。定价135美元,给SpaceX带来了约1.77万亿美元的估值。Howard Marks有个很好的框架——IPO这天往往是"卖方最优先"的时刻,买方需要额外审慎。

[小晴] 上市首日收盘价冲到了160.95美元,全天交易区间在149到176美元之间震荡。我在Reddit和X上看到大量散户在讨论这票,有人非常亢奋,也有人在说"估值已经在全球第七大公司了,再买要去哪?"

[阿哲] 基本面支撑方面,SpaceX在S-1文件里披露了Terafab芯片制造计划,长期目标是每年生产一太瓦的算力硬件——这意味着它不只是火箭公司,更是算力基础设施的潜在玩家。但我会提醒大家:Starlink的盈利模型已经相对清晰,但芯片和深空业务的DCF怎么建模,当前还是高度不确定的。另外SpaceX并购了马斯克的AI公司xAI,这让公司的财务报表变得更加复杂。短期内SPCX更多是情绪和筹码博弈,机构定价需要时间。

[小晴] 第二个想提的是CRWV——CoreWeave。这票在Reddit和Stocktwits上的热度持续在高位,特别是今天PLTR小跌之后,有人开始把CRWV当作"更便宜的AI基础设施替代"来讨论。CoreWeave的业务模式是提供以GPU和CPU算力为核心的云平台,支持AI模型训练、推理、以及数据集优化,面向从大型企业到初创公司的各类算力密集型需求方。

[阿哲] 结构上我有一个保留意见。有深度研究者指出,CoreWeave高度依赖高杠杆硬件采购的商业模式,跟成熟的超大规模云厂商相比,面临相当显著的结构性逆风。它的增长速度是真实的,但债务扩张的速度同样惊人。Damodaran在讲"成长型公司估值"时反复强调:高速成长可以支撑高估值,但前提是资本再投入能产生足够高的回报率——CoreWeave这个问题现在还没有被市场充分定价。

[小晴] 第三个是ASTS——AST SpaceMobile。这票跟RKLB一样,SpaceX上市当天ASTS也下跌了约10%,但Reddit上的社区反应很有意思——很多人把它理解成"被误伤的买入机会",在跌后开始大量讨论。

[阿哲] ASTS的逻辑是直连卫星手机信号——跟运营商合作,让普通智能手机不用任何额外硬件就能直接连卫星网络。这个市场如果能打开,TAM非常大。但它也面临资本密集、执行风险高、竞争格局不明确的多重问题。所以我对ASTS的定性仍然是"高弹性、高风险的主题赛道票"。

[小晴] 还有一个今天在财经社媒上出现得比较多的名字是VOYG——Voyager Technologies。它在SpaceX上市前后被频繁提及,是另一家国防+航天方向的新上市公司,主要做国家安全领域的轨道和高超音速系统。这个票目前流通盘很小、机构覆盖极少,更多是投机性质。我会把它归到"拿着放大镜观察"的名单里,但暂时不会真正进入持仓讨论。

[阿哲] 好,总结一下今天的扫描。从宏观背景来看,SpaceX上市是本周市场最重要的结构性事件——它一方面抽走了部分空间赛道的公开市场流动性,另一方面也在重新定价AI与算力基础设施叙事的天花板。持仓单里的多数票今天的回调更多是情绪性的,基本面并未发生实质性变化。

[小晴] 对,需要真正注意的是两个时间节点:MU的6月24日财报,以及RKLB能否在SpaceX上市后的资金重新定价中找到新的均衡位。TSLA今天是少数跑赢的票,SpaceX的光环效应还没有退潮,下周初可以继续观察。

[阿哲] Howard Marks有句话很适合今天的市场状态——"当每个人都在看同一个故事的时候,最危险的地方往往就是这个故事本身的中心。"SPCX今天在聚光灯下,反而需要更冷静。下周见。
